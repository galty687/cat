在 Trados中使用机器翻译

前提条件

  1. 安装MT Enhanced Plugin for Trados Studio插件

  2. 申请微软 translation api 账号

具体步骤

  1. 创建项目的时候添加MT

  2. 在MT中输入translation API key

  3. 选择订阅所属的区域

4. 在翻译过程即可由机器翻译自动完成译文
