# 在 Trados中使用机器翻译



## 前提条件

1. 安装[MT Enhanced Plugin for Trados Studio](https://appstore.sdl-china.cn/cn/language/app/mt-enhanced-plugin-for-trados-studio/604/)插件
2. 申请微软 [translation api](https://www.microsoft.com/en-us/translator/business/translator-api/) 账号



## 具体步骤

1. 创建项目的时候添加MT
2. 在MT中输入translation API key
3. 选择订阅所属的区域
